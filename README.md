# English Grammar App 📚✨

مرحباً! هذا هو **تطبيق قواعد اللغة الإنجليزية** التفاعلي، يحتوي على أكثر من **100 سؤال وإجابة مفصلة**.  
التطبيق مصمم ليعمل على المتصفحات الحديثة ويمكن فتحه على **الآيفون، الآيباد، وأي جهاز** بسهولة.  

---

## 🖤 تصميم التطبيق
- واجهة **سوداء مطفيّة** مع إطار **ذهبي** أنيق  
- قائمة **الاقتراحات** لإضافة أي اقتراح من المستخدم  
- قائمة **الأسئلة المتاحة** لاستعراض الأسئلة الكاملة من 1 إلى 101+  
- زر **تجربة** لكل سؤال لعرضه في الحقل الرئيسي  
- زر **عرض الإجابة** يظهر الإجابة المفصلة للسؤال المختار  

---

## 🚀 كيفية الاستخدام
1. افتح الرابط الخاص بـ GitHub Pages (أو أي متصفح حديث)  
2. اضغط على **الأسئلة المتاحة** لاستعراض الأسئلة  
3. اضغط **تجربة** لعرض السؤال في الصفحة الرئيسية  
4. اضغط **عرض الإجابة** لمشاهدة الإجابة المفصلة  
5. أضف اقتراحاتك في **الاقتراحات**، وستظهر على الصفحة  

---

## 📂 الملفات
- `index.html` → يحتوي على التطبيق بالكامل (HTML + CSS + JS)  
- **ملفات إضافية (اختياري)**: أيقونات، صور، أو CSS منفصل إذا أردت تعديل التصميم  

---

## 🌐 نشر التطبيق
لتشغيل التطبيق على الآيفون أو أي جهاز:
1. افتح الرابط المباشر من GitHub Pages  
2. اضغط **Share → Add to Home Screen**  
➡️ التطبيق سيظهر كأيقونة في الشاشة الرئيسية تماماً مثل أي تطبيق آخر  

---
